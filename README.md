# Machine Learning Course Projects

Two independent ML projects completed as part of a university machine learning course.

## Projects

### 1. Abalone Age Prediction — `abalone_age_prediction/`

**Type:** Pair project (with [Katsiaryna Schastnaya](https://github.com/))

Supervised regression task: predicting the age of abalone (marine mollusks) from physical measurements, as a non-invasive alternative to counting shell rings under a microscope.

→ See [`abalone_age_prediction/README.md`](./abalone_age_prediction/README.md) for details.


### 2. NYC Airbnb Price Prediction — `nyc_airbnb_price_prediction/`

**Type:** Individual project (topic assigned by instructor)

Supervised regression task: predicting Airbnb listing prices in New York City based on location, room type, and host activity features.

→ See [`nyc_airbnb_price_prediction/README.md`](./nyc_airbnb_price_prediction/README.md) for details.


## Tech Stack

Python, pandas, scikit-learn, matplotlib, seaborn, SHAP, Streamlit, joblib
